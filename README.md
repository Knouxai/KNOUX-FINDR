# KNOUX FINDR - محرك البحث المحلي المدعوم بالذكاء الاصطناعي

🚀 **تطبيق سطح مكتب متقدم للبحث في الملفات مع ذكاء اصطناعي مدمج**

## الميزات الرئيسية

### 🤖 الذكاء الاصطناعي المدمج (Offline AI)

- **تحليل المحتوى الذكي**: استخراج وتحليل النصوص من جميع أنواع الملفات
- **التصنيف التلقائي**: تصنيف ذكي للملفات حسب المحتوى والسياق
- **البحث المحسن بالـ AI**: نتائج بحث أكثر دقة وصلة
- **الاقتراحات الذكية**: اقتراحات بحث تلقائية بناءً على السياق
- **دعم اللغة العربية والإنجليزية**: معالجة نصوص متقدمة للغتين

### 📁 فهرسة الملفات المتقدمة

- **فهرسة تلقائية**: فهرسة جميع الملفات في المجلدات المحددة
- **مراقبة الملفات**: تتبع التغييرات في الوقت الفعلي
- **دعم أنواع ملفات متعددة**: PDF, Word, Excel, PowerPoint, الصور, الفيديو, والمزيد
- **استخراج البيانات الوصفية**: معلومات تفصيلية عن كل ملف
- **حفظ التاريخ**: تتبع تاريخ الإنشاء والتعديل والوصول

### 🔍 البحث المتقدم

- **البحث النصي الكامل (FTS)**: بحث سريع في محتوى الملفات
- **البحث بالمرشحات**: تصفية حسب النوع، التاريخ، الحجم، التصنيف
- **البحث الدلالي**: فهم المعنى وليس فقط الكلمات المطابقة
- **البحث المتعدد اللغات**: دعم العربية والإنجليزية
- **نتائج مرتبة**: ترتيب النتائج حسب الصلة والأهمية

### 🛠️ أدوات إدارة الملفات

- **كشف الملفات المكررة**: العثور على الملفات المتطابقة وإدارتها
- **التنظيم الذكي**: اقتراحات لتنظيم وتصنيف الملفات
- **الأمان والخصوصية**: جميع العمليات محلية بدون إنترنت
- **النسخ الاحتياطي الذكي**: تحديد الملفات المهمة للنسخ الاحتياطي

## التقنيات المستخدمة

### Backend AI & File Processing

- **Node.js**: منصة الخادم الأساسية
- **Electron**: تطبيق سطح المكتب متعدد المنصات
- **SQLite3**: قاعدة بيانات محلية للفهرسة
- **node-nlp**: معالجة اللغات الطبيعية
- **compromise**: تحليل النصوص الإنجليزية
- **franc**: كشف اللغة التلقائي

### File Processing

- **pdf-parse**: استخراج النص من ملفات PDF
- **mammoth**: استخراج النص من ملفات Word
- **sharp**: معالجة ومعلومات الصور
- **chokidar**: مراقبة تغييرات الملفات
- **mime-types**: كشف أنواع الملفات

### Frontend

- **React 18**: واجهة المستخدم التفاعلية
- **Tailwind CSS**: تصميم متجاوب وحديث
- **Glassmorphism UI**: تصميم زجاجي عصري
- **RTL Support**: دعم كامل للغة العربية

### Search & Indexing

- **SQLite FTS5**: البحث النصي الكامل
- **Fuse.js**: البحث الضبابي
- **Custom AI Ranking**: ترتيب النتائج بالذكاء الاصطناعي

## التثبيت والتشغيل

### متطلبات النظام

- Node.js 16+
- npm أو yarn
- نظام تشغيل: Windows, macOS, أو Linux

### خطوات التثبيت

1. **تحميل المكتبات:**

```bash
npm install
```

2. **بناء التطبيق:**

```bash
npm run build
```

3. **تشغيل تطبيق سطح المكتب:**

```bash
npm run electron
```

4. **تطوير مع Hot Reload:**

```bash
npm run electron-dev
```

5. **إنشاء حزمة للتوزيع:**

```bash
npm run package
```

## الاستخدام

### الإعداد الأولي

1. عند أول تشغيل، سيبدأ التطبيق بفهرسة المجلدات الأساسية
2. يمكنك إضافة مجلدات إضافية من قائمة "ملف" > "فهرسة مجلد جديد"
3. سيعمل الذكاء الاصطناعي على تحليل وتصنيف الملفات تلقائياً

### البحث المتقدم

- **البحث البسيط**: اكتب أي كلمة أو عبارة
- **البحث بالمرشحات**: استخدم الفلاتر في الشريط الجانبي
- **البحث بالـ AI**: فعّل "البحث بالذكاء الاصطناعي" للنتائج المحسنة
- **البحث بالتصنيف**: ابحث ضمن تصنيف محدد

### الميزات الذكية

- **التصنيف التلقائي**: من قائمة "ذكاء اصطناعي" > "تصنيف تلقائي"
- **تحليل المحتوى**: اختر "تحليل المحتوى" لتحليل شامل
- **كشف المكررات**: استخدم زر "البحث عن المكررات"
